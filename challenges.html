Challenge 1: Social Media Moderator

You have been assigned to moderate a social media platform where certain words are not allowed in user posts. Build a tool that checks a message and censors restricted words, making the post safe to share.

This tool should be able to do the following tasks:

  • Analyze the message entered by the user and detect restricted words.
  • Keep a count of restricted words found and inform the user about it.
  • Censor the restricted words in the post by replacing them with ***.
  • The user should not be allowed to post if restricted words are more than 3.
